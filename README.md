# 💊 LabesPharmacy — صيدلية لاباس

A modern, bilingual (French/Arabic) pharmacy website built with **React**, **TypeScript**, and **Tailwind CSS**. Designed with a Tunisian cultural identity — featuring arabesque geometric patterns, a dark blue/turquoise theme, and bilingual UI.

![LabesPharmacy Preview](https://i.ibb.co/rRXHc2Cb/hero-section-phar.png)
![LabesPharmacy Preview](https://i.ibb.co/yBcVgf7H/Screenshot-2026-03-19-024121.png).

---

## ✨ Features

- 🌙 **Dark theme** with blue & turquoise Tunisian-inspired palette
- 🕌 **Arabesque SVG patterns** & zellige geometric dividers
- 🌍 **Bilingual UI** — French + Arabic throughout
- 📱 **Fully responsive** — mobile, tablet, desktop
- ⏰ **Live open/closed indicator** based on real-time clock
- 🎞️ **Smooth animations** powered by Framer Motion
- 📬 **Contact form** with clean validation UI
- 📰 **Health tips blog** section
- 🗺️ **Map & contact info** section

## 🚀 Getting Started

### Prerequisites
- Node.js v18+
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/mohamedflifel/labes-pharmacy.git

# Navigate into the project
cd labes-pharmacy

# Install dependencies
npm install

# Start the development server
npm run dev
```

Then open [http://localhost:5173](http://localhost:5173) in your browser.

## 🎨 Design Decisions

- **Dark slate background** gives a premium, modern feel while being easy on the eyes
- **Cyan/teal accent colors** reference the blue tones of Tunisian zellige tilework and Mediterranean sea
- **Georgia serif font** for headings gives an elegant, timeless feel fitting for a healthcare brand
- **Bilingual labels** (FR + AR) reflect the real-world context of Tunisian businesses
